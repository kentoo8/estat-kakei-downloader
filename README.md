# 家計調査 月次支出データ取得CLI

e-Stat APIを使用して、家計調査（2025年改定）の月次支出データを取得するCLIツールです。

## セットアップ

1. 依存パッケージをインストール

```bash
pip install -r requirements.txt
```

2. `.env`ファイルを作成し、APIキーを設定

```bash
cp .env.example .env
```

`.env`を編集して`ESTAT_APP_ID`に[e-Stat](https://www.e-stat.go.jp/)で取得したアプリケーションIDを設定してください。

## 使い方

```bash
python cli.py
```

## ファイル構成

| ファイル | 説明 |
|----------|------|
| `cli.py` | メインCLI |
| `data_fetcher.py` | e-Stat API通信 |
| `kakei_2025_cache.json` | 品目キャッシュ（689品目） |
| `品目一覧_2025年改定.md` | 品目一覧（人間用） |

## 仕様

- **対象データ**: 家計調査 2025年改定 月次データ
- **デフォルト条件**: 二人以上の世帯、全国、全期間
- **出力形式**: CSV（UTF-8 BOM付き）
- **保存先**: `DATA_SAVE_DIR`で指定（デフォルト: `./data`）
